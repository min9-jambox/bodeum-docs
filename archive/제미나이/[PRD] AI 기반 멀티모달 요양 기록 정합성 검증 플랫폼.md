# [PRD] AI 기반 요양 기록 정합성 검증 및 표준 데이터 플랫폼

## 1. 개요 (Overview)
* **제품명:** Care-Sync Intelligence
* **목표:** 요양 현장의 비정형 데이터(음성, 이미지)를 실시간으로 분석하여 요양 기록의 신뢰도를 보증하고, 표준화된 디지털 증거를 생성하는 요양 인프라 솔루션.
* **배경:** 요양보호사의 기록 업무 과부하 해소 및 고령/외국인 종사자의 행정 오류 방지, 나아가 공단 실사 시 환수 리스크를 원천 차단하는 관리 시스템.

## 2. 사용자 페르소나 (Detailed Personas)
* **국내 고령 보호사:** 스마트폰 UI 조작이 서툴러 '말하기와 촬영' 기반의 제로-UI(Zero-UI)가 필수적인 숙련된 돌봄 전문가.
* **외국인 보호사:** 한국어 행정 용어 및 문장 작성에 어려움을 겪으나, 도메인 지식은 보유한 전문 연수 인력.
* **관리자(센터장):** 공단 실사 대응, 부정수급 리스크 관리, 업무 효율화를 통해 운영 수익을 극대화해야 하는 기관장.
* **보호자:** 부모님의 일일 상태(식사, 기분, 거동 등)를 실시간 시각 데이터로 확인하고 안심하고 싶은 서비스 수요자.

## 3. 핵심 기능 요구사항 (Requirements)

| 우선순위 | 기능명 | 상세 기능 및 스펙 | 비고 |
| :--- | :--- | :--- | :--- |
| **P0** | 멀티모달 입력 시스템 | 음성(모국어) + 현장 사진 동시 캡처 및 타임스탬프 동기화 | Zero-UI |
| **P0** | 정합성 검증 엔진 | 사진(객체 인식)과 음성(기록)의 논리적 모순 감지 및 수정 요청 | 핵심 특허 |
| **P0** | 표준 데이터 저장소 | W4C(요양급여 표준) 규격에 맞춰 정형 데이터로 변환/보관 | Export-Ready |
| **P1** | 보호자 알림톡 엔진 | AI 분석 기반 '감성 리포트' 자동 생성 및 카카오톡 연동 | 안심 서비스 |
| **P1** | 도메인 엔티티 매핑 | 요양 표준 용어(표준급여 기준) 기반 문장 자동 완성 | 자동 행정화 |
| **P1** | **글로벌 확장 엔진** | 일본(Kaigo), 독일(DiPA) 표준 규격 대응용 공통 데이터 모델(CDM) | 확장성 |
| **P2** | 관리자 대시보드 | 보호사별 기록 정확도 모니터링 및 리스크 태깅 | 운영 효율 |

## 4. 시스템 아키텍처 및 데이터 흐름



1. **Edge Layer (수집):** 단말 내 비식별화 및 데이터 패키징.
2. **Middleware (결정론적 필터링):** 30년 차 엔지니어의 로직으로 LLM의 할루시네이션(환각)을 제거.
3. **Cognitive Layer (분석):** Claude 3.5 기반 요양 전문 도메인 맥락 분석.
4. **Data Layer (저장):** W4C 규격 준수 표준 데이터 저장소 (API 연동 대비).
5. **Output Layer (전송):** 보호자용 감성 알림톡 및 센터용 행정 리포트 생성.

## 5. 비기능적 요구사항
* **정확도:** 정합성 검증 엔진을 통한 기록 오류(수가 환수 리스크) 99.8% 예방.
* **사용성:** 60대 이상 고령자 기준 10초 이내 기록 완료.
* **보안성:** 모든 개인 데이터는 로컬 처리 및 클라우드 전송 전 비식별화.

## 6. 성공 지표 (Success Metrics)
* **행정 효율:** 요양 기록 작성 시간 80% 이상 단축.
* **리스크 방어:** 데이터 정합성 검증을 통한 부정수급 의심 사례 사전 필터링 건수.
* **만족도:** 알림톡 발송에 따른 보호자 피드백 확인율 및 요양기관 센터장 재계약률.
* **글로벌 확장성:** 공통 데이터 모델(CDM)을 통한 타 국가(일본, 독일) 시스템 이식성 평가.

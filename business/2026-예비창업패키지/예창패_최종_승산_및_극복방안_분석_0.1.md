# 🏆 예비창업패키지 승산 진단 및 약점 극복 전략 (최종 업데이트판)

## 1. 합격 가능성 종합 진단: "서류 통과는 가능, 최종 선정은 조건부"

### 현재 상태 점수 (100점 만점)

| 평가 항목 | 배점 | 현재 점수 | 문제점 |
|:---|:---:|:---:|:---|
| 문제인식 명확성 | 25점 | **22점** | 강점. 팩트 기반 서술 우수. |
| 실현가능성 (기술+실행) | 35점 | **22점** | 고객 인터뷰 없음, MVP 범위 과다. |
| 성장전략 (BM+시장진입) | 25점 | **18점** | BM 논리 우수하나 검증 증거 없음. |
| 팀 구성 및 역량 | 15점 | **10점** | 1인 팀, 도메인 경험 미약. |

> **핵심 진단**: 아이디어 자체는 좋다. 문제는 **"이렇게 될 것이다"는 논리만 있고, "이미 확인했다"는 증거가 없다**는 것이다.

## 2. 강점 — 이것은 반드시 유지한다

### ① 날카로운 문제 정의 (니치 마켓 선점)
"60대 이상 66.1% 요양보호사의 스마트폰 타자 입력 불가 문제"라는 한 문장 팩트가 매우 강력하다. 심사위원이 즉각 고개를 끄덕이는 수준의 문제 정의다.

### ② 기존 ERP와 공존하는 포지셔닝
"이지케어를 버리지 말고, 우리 걸 얹으세요"라는 Add-on 전략은 현실적이다. 1인 개발자가 726억 매출 기업과 싸우지 않는 전략이다.

### ③ 구체적인 단위 경제성
센터당 공헌이익 7,200원, BEP 800개소(점유율 3.5%)라는 숫자는 심사위원이 원하는 재무 설득력이다.

### ④ ESG 스토리
요양보호사 노동 인권 + 노인 인권 + 요양 인력 공급 안정화라는 세 겹의 사회적 가치는 예창패와 이후 소셜임팩트 투자에서 강력한 가점 요소다.

## 3. 치명적 약점 3가지 — 이것이 탈락의 이유

### ⚠️ 약점 1: 고객 인터뷰·수요 검증 증거 전무 (가장 치명적)

**문제**: 사업기획서 전체에 실제 원장님·요양보호사와 나눈 인터뷰가 단 한 건도 없다. "원장님이 결제할 것이다"는 논리적 추론만 있고, "실제로 10명에게 물어봤더니 7명이 쓰겠다고 했다"는 증거가 없다.

**해결 방법** (지금 당장 가능):
- 주변 요양원 원장님 10~15명 연락, 15분 인터뷰 실시
- 질문: ① 요양보호사 기록 업무에서 가장 힘든 것은? ② 월 1만 원에 이 기능 쓸 의향이 있나요?
- 결과를 아래 형식으로 사업계획서에 삽입:

```
[현장 인터뷰 사례]
• 서울 은평구 방문요양센터 원장 (수급자 18명, 2026년 3월)
  "요양보호사들이 관찰일지 쓰기 싫다고 그만두는 경우가 실제로 있다.
   말만 하면 AI가 써준다면 월 1만 원이면 당장 쓰겠다."
• 인천 부평구 방문목욕 센터 요양보호사 (경력 7년, 60대, 2026년 3월)
  "물 만지고 나서 스마트폰 치는 게 제일 힘들어요. 
   말로 하면 된다니까 진짜요?"
```

---

### ⚠️ 약점 2: MVP 범위가 너무 넓어 실현 가능성 의심

**문제**: 현재 기획서에 나열된 MVP 기능 목록:
- React Native 요양보호사 앱 (iOS/Android)
- 원장님용 웹 백오피스
- GPS 지오펜싱 엔진
- Whisper STT 파이프라인
- GPT-4o-mini 연동 텍스트 정제
- 카카오 알림톡 API 연동
- 멀티모달 정합성 검증 엔진
- W4C 표준 필드 자동 매핑

이것을 1인이 4개월 안에 완성한다는 것은 심사위원이 쉽게 믿지 않는다.

**해결 방법**: MVP 범위를 1단계/2단계로 명확히 쪼개라.

```
1단계 MVP (2,000만 원, 4~5개월): 반드시 완성
  - 요양보호사 음성 녹음 앱 (단일 버튼 화면)
  - 음성→Whisper STT→GPT-4o-mini→카카오 알림톡 단일 파이프라인
  - 원장님 웹: 스케줄 엑셀 업로드 + AI 일지 확인 + ERP 호환 엑셀 다운로드
  - 5개 센터 시범 운영 + 데이터 수집

2단계 (2,000만 원, 중간평가 통과 후): 추가 개발
  - GPS 지오펜싱 출퇴근 넛지
  - 프리미엄 마케팅 리포트
  - 멀티모달 정합성 검증 (특허 연계)
  - 100개소 확장 영업
```

---

### ⚠️ 약점 3: 팀 구성의 취약성

**문제**: 1인 팀 + 요양/복지 도메인 전문성 미약

**해결 방법**:
1. 솔직하게 약점을 인정하고 보완 방안을 명시
2. 요양원 관련 자문위원 1명이라도 이름을 올려라 (사회복지사 출신 지인, 현직 원장님 등)
3. "1인 팀의 강점"(외주 없이 예산 100% 집중, 의사결정 속도)을 오히려 어필

## 4. 기회 요인 — 이것을 더 강조하라

### ① 2026년 예창패 공고 타이밍 완벽
- 3월 6일~24일 접수, 지금이 적기
- 초고령화 사회 + 요양 인력난 이슈가 사회적으로 가장 뜨거운 시점

### ② 케어로그가 오히려 우리의 증거
케어로그가 구글 창구 7기에 선정됐다는 것 = 이 시장에 니즈가 있다는 객관적 증거. "케어로그도 같은 방향을 보고 있지만, 저희는 그들이 해결 못한 타자 입력 문제에 집중합니다"라고 포지셔닝하면 오히려 시장 검증 근거가 된다.

### ③ 2026 예창패 구조 변화 이해
- 1단계 2,000만 원 → 중간평가 → 통과 시 2단계 2,000만 원
- 1단계에서 MVP를 좁게 잡고 실제로 완성하는 것이 합격의 열쇠

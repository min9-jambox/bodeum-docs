## 1. 전체 메뉴 구조

패밀리케어 ERP는 **재가 장기요양기관** 운영에 필요한 전 업무를 커버하는 올인원 SaaS다.
좌측 사이드바 기반의 웹 애플리케이션으로, 9개 대메뉴 아래 약 40개 이상의 세부 화면으로 구성된다.

| 대메뉴 | 하위 기능 수 | 핵심 역할 |
|--------|-----------|---------|
| 홈(대시보드) | 1 | 일일 방문일정, 수급자 현황 그래프, 기초평가 미실시 알람 |
| 수급자 관리 | 7 | 수급자 정보, 기초평가, 상태변화 기록지, 상담일지, 안전관리 자료 |
| 일정관리 | 10+ | 수급자·보호사·복지사 일정, 라운딩, 알림톡 발송, 공단연동 |
| 급여제공 | 4 | 청구내역상세, RFID 급여제공기록, 공단 일정계획 연동 |
| 직원관리 | 7 | 직원정보, 근무일지/출근부, 고충처리, 사례회의록, 회의록, 건강검진 |
| 직원 월 급여 | 6 | 급여 기초설정, 월 급여대장, 명세서 발송, 연간급여대장, 퇴직적립금 |
| 본인부담금 | 3 | 청구관리, 입금관리, 영수증/납부확인서 |
| 운영관리 | 6 | 기관정보 설정, 차량/욕조/비품관리, 보험 현황, 메뉴 권한 |
| 부가서비스 | 2 | 서식자료실, 업무 가이드(일일/주간/월간) |

## 2. 화면별 상세 분석

### 2.1 홈 대시보드

**구성 요소:**
- 일일 방문일정 알림 (당일 방문 예정 수급자 목록)
- 시설 현황 바 그래프 (수급자 현황 추이)
- 수급자 현황 그래프 (월별 추이)
- 기초평가 미실시대상 알람 (놓치기 쉬운 평가 일정 자동 알림)
- 급여비교 메뉴 미선언 항목 (설정 누락 경고)
- 공지사항

**분석:** 대시보드가 단순한 현황 표시를 넘어, "놓치기 쉬운 업무"를 프로액티브하게 알려주는 알람 기능이 핵심 가치다. 기관 평가 대비를 위한 행정 누락 방지 용도.

---

### 2.2 수급자 관리

#### 수급자 정보관리
- 좌측 수급자 리스트 + 우측 상세 정보 패널 (마스터-디테일 레이아웃)
- **기본정보**: 이름, 생년월일, 성별, 장기요양 등급, 인정번호, 인정유효기간
- **사진**: 수급자 프로필 사진 등록
- **기초정보**: 주소, 연락처, 보호자 정보
- **급여 이용 현황**: 현재 이용 중인 급여 서비스 종류
- **급여 이용 기록**: 과거 이력

#### 전체 기초평가 현황
- 수급자별 평가 시기, 평가 종류, 복지사명, 인정만료일 일괄 조회
- 기간 필터, 등급·복지사 필터

#### 기초평가 관리
- 현황 선택 드롭다운: 계약해지 / 보류 / 수급중
- 나이, 등급, 복지사, 인정만료 기준 정렬
- 수급자가 없으면 "현재 등록된 수급자가 없습니다" 안내

#### ⭐ 상태변화 기록지 작성
- **보듬의 핵심 타겟 화면**
- 날짜별·수급자별 서술형 기록 입력
- 현재 UI: 단순 텍스트 입력 폼 (수기 입력)
- 기록 이력 테이블 (날짜, 복지사, 수급자)
- **한계점**: AI 자동 작성 기능 없음. 수기 타이핑 필요. 이 부분이 보듬의 핵심 차별점.

#### 상담일지
- 날짜, 상담유형, 상담자, 수급자, 상담내용, 비고 테이블
- 상담일지 → 급여제공반영 연계 기능 (상담 결과를 급여 계획에 직접 반영)
- 급여제공반영 작성 모달: 수급자 정보, 급여변경 정보(급여종류, 횟수, 시간, 급여내용, 급여비용) 입력

#### 수급자 안전관리 자료제공
- 수급자별 안내자료 제공 이력 관리
- 제공내역 등록/출력 기능
- 안전관리 자료 출력 (기관 평가용 서류)

---

### 2.3 일정관리

#### 수급자 일정관리
- 좌측 수급자 리스트 + 우측 상세 일정 패널
- **급여 이용 계획**: 요일별·시간별 급여 계획 매트릭스
- **급여 이용 계획 수정**: 시간/요일 단위로 계획 변경
- **급여비용 상세정보**: 팝업으로 등급별 급여비용 자동 산출
- **급여비용 내역비교**: 예상 비용 vs 실제 비용 비교
- **일정 알림톡 발송**: 🔑 수급자/보호자에게 월별 일정을 카카오 알림톡으로 일괄 발송. 수급자명, 급여종류, 수급자 전화번호, 발송 유무 확인 후 발송.

#### 요양보호사 근무 일정
- 캘린더 UI (월간 뷰)
- 보호사별 근무일정 등록, 근무일정 보내기(알림톡)

#### 보호사 일정 한눈에 보기
- 일/주/월 3가지 뷰 전환
- 방문요양 / 방문간호 / 방문목욕 색상 구분 범례

#### 복지사 담당수급자 지정
- 복지사 ↔ 수급자 매핑 관리

#### 복지사 라운딩 일정 / 라운딩 일정 한눈에 보기
- 복지사 방문(라운딩) 일정 캘린더
- 일/주/월 뷰 전환

#### 복지사 업무수행일지 작성
- 캘린더 기반 일지 작성
- 날짜 선택 → 업무수행 내용 기록

#### 일정계획 및 청구내역상세 공단연동
- **공단 데이터 직접 연동** 기능
- 공단 일정계획 연동방법 안내 포함
- 업로드/다운로드 양방향 연동

---

### 2.4 급여제공

#### 청구내역상세
- 수급자별 급여제공 내역 전체 조회
- 컬럼: 일자, 수급자명, 등급, 본인부담률, 급여동률, 요양보호사, 급여 제공시간, 구분, 원거리, 인지 활동, 급여 기록
- **급여비용 내역 비교**: 급여 제공 내역과 비용 교차 검증
- **청구내역 상세 업로드**: 공단 일정계획 연동

#### 급여제공기록 RFID
- **공단 RFID 연동**: NFC/RFID 태그 기반 출퇴근 인증 연동
- 급여제공기록 입력 모달 (4단계):
  - 1단계: 기본정보 (일자, 급여종류)
  - 2단계: 시작/종료시간, 급여내용 선택
  - 3단계: 추가 서비스 내용
  - 4단계: 확인 및 저장
- 급여제공기록표 및 방문요양 활동기록지 연계

---

### 2.5 직원관리

#### 직원 정보관리
- 마스터-디테일 레이아웃
- **기본정보**: 직원명, 생년월일, 성별, 연락처, 담당직무
- **사진** 등록
- **4대 보험 수정** 모달: 국민연금, 건강보험, 고용보험, 산재보험 각각 취득일/상실일 관리
- **배상책임보험** 수정 + 이력보기
- **직원 상태 이력 변경**: 재직/퇴직/휴직 상태 전환 이력
- **직원정보출력**: PDF/인쇄

#### 근무일지 및 출근부 관리
- 캘린더 기반 근무일정 뷰
- **근무일정 일괄등록**: 다수 직원 근무일정 한 번에 등록
- **근무일정 관리**: 개별 수정, 수정 설명 기능
- **출근부 전체 출력**: 2페이지 분량 인쇄용 출근부

#### 고충처리 관리
- 접수일자, 신청인, 고충사항, 처리결과, 처리일자, 처리자, 첨부파일, 수정
- 직원 선택 시 현황별/담당직종별 필터

#### 수급자 사례회의록
- 사례관리 회의록 양식 (회의 주제, 참석자, 내용)
- 관련 자료 첨부 기능
- 기관 평가 필수 서류

#### 직원회의록(회의, 교육)
- 직원회의록 / 직원교육내용 / 복지 및 포상 3개 탭
- 월별 회의 목록
- 회의록 양식: 참석자, 주제, 내용

#### 건강검진관리
- 직원별 건강검진 이력 (검진일, 담당직종, 최근 검진 제출일, 미착성 여부)

---

### 2.6 직원 월 급여

#### 직원 월 급여 기초 설정
- 직원 목록 + 계약정보 수정
- **시급제**: 시급, 주휴수당 설정, 근무유형별 시간 설정
- **월급제**: 기본급, 수당 항목별 금액 설정
- 4대 보험료 자동 계산

#### 시설 월 급여 기초 설정(수당/공제)
- **수당 항목 관리**: 수당항목 신규등록 (이름, 과세구분, 적용방식)
- **공제 항목 관리**: 공제항목 신규등록
- **시설 급여 설정**: 기관 전체에 적용되는 급여 기본 설정 (2페이지 분량)

#### 월 급여대장(급여명세서)
- 전 직원 급여 테이블 (기본급, 각종 수당, 공제, 실지급액)
- **급여 지급일 설정**: 매월 지급일 지정
- **명세서 발송**: 급여명세서 카카오 알림톡/문자 발송

#### 직원 연간급여대장
- 1~12월 급여 합산 연간 대장
- 전체 출력 (2페이지)

#### 퇴직적립금 관리
- 월간 적립금 목록
- 퇴직적립금 목록 출력

---

### 2.7 본인부담금

#### 본인부담금 청구관리
- 수급자별 청구 내역 (급여종류, 횟수, 급여비용, 공단부담, 본인부담, 비급여, 청구금액)
- **청구서 일괄 생성**: 전체 수급자 대상 일괄 청구서 생성

#### 본인부담금 입금관리
- 수급자별 입금 현황 (납부금액, 미수금액)
- **본인부담금 청구내역** 테이블 (연월별 청구/수납 추적)
- **선납이력**: 선납금 관리
- **영수증 발송**: 카카오 알림톡으로 영수증 발송
- **장기요양급여비 납부확인서 출력**: PDF/인쇄 (2페이지)
- **환불처리**: 과납 환불 프로세스

---

### 2.8 운영관리

#### 기관정보 설정
- **방문요양**: 기관 기본정보, 등급별 재가급여 월한도액, 본인 부담금요율, 수급자 미적용 급여비에 본인부담부여 여부 설정
- **방문목욕**: 차량별 목욕 설정
- **방문간호**: 시간별 급여비용 설정
- **이력보기**: 급여비용 변경 이력

#### 차량/욕조/비품관리
- **차량관리**: 차량정보, 운행일지
- **욕조 소독일지**: 소독일, 수급자, 소독시간, 소독자
- **간호 비품관리대장**: 비품 등록, 재고 관리
- **간호기구(소독)관리**: 소독 이력

#### 배상책임보험 가입 현황
- 보험 상품명, 보험증권번호, 보험회사, 청약일, 가입 직원 목록

#### 메뉴 권한 설정
- 직원별 메뉴 접근 권한 관리

---

### 2.9 부가서비스

#### 서식자료실
- 기관 운영에 필요한 각종 서식(양식) 다운로드
- 서식명, 자료종류, 조회수, 다운로드 기능
- 카테고리: 기관 운영서식, 수급자케어관련 양식 등

#### 업무 가이드
- **일일 업무(주간 업무)**: 급여제공기록 RFID, 수급자 일정변경 반영 체크, 복지사 라운딩 정리, 근무일지 및 출근부 관리, 수급자 상담관련, 고충처리 관리
- **주간 확인사항**: 상담일지, 복지사 업무수행일지
- **월간 업무**: 급양보호사 근무 일정, 복지사 라운딩 일정, 급여비교 메뉴 미선언 항목 확인, 직원 급여 확인, 본인부담금 관리(청구/입금)

## 3. UX/UI 특성 종합

| 항목 | 평가 |
|------|------|
| 레이아웃 | 좌측 사이드바 + 컨텐츠 영역. 마스터-디테일 패턴 다수 사용 |
| 색상 체계 | 녹색(#2E7D32 계열) 기반. 버튼·링크·헤더 모두 녹색 통일 |
| 캘린더 뷰 | 일정관리 전반에 일/주/월 전환 캘린더 적극 활용 |
| 모달/팝업 | 급여제공기록, 4대보험, 알림톡 발송 등 상세 작업은 모달 팝업 |
| 인쇄/출력 | 거의 모든 관리 화면에 PDF/인쇄 출력 버튼 존재 |
| 알림톡 연동 | 일정 알림, 급여명세서, 영수증 등 카카오 알림톡 발송 내장 |
| 공단 연동 | RFID, 일정계획, 청구내역 등 공단 데이터 양방향 연동 |
| 반응형 | 데스크톱 웹 기반, 모바일 최적화는 별도 앱(패밀리노트) 담당 |

## 4. 패밀리케어 ERP의 강점과 약점

### 강점
1. **올인원 커버리지**: 수급자관리부터 급여/세무까지 기관 운영 전 업무 커버
2. **공단 연동**: RFID, 일정계획, 청구내역 등 공단 시스템과 직접 연동
3. **알림톡 내장**: 일정, 급여명세서, 영수증 등 다양한 알림톡 발송 기능
4. **업무 가이드**: 일일/주간/월간 업무 체크리스트로 행정 누락 방지
5. **인쇄/출력**: 기관 평가 대비 각종 서류 인쇄 기능 충실

### 약점
1. **⭐ AI 부재**: 상태변화 기록지, 상담일지 등 서술형 기록을 수기 타이핑해야 함
2. **⭐ 음성 입력 없음**: 모든 데이터 입력이 키보드 기반
3. **UI 노후화**: 클래식한 테이블 중심 UI, 모던 UX와 거리 있음
4. **모바일 웹 미최적화**: 데스크톱 전용 레이아웃
5. **데이터 분석 미흡**: 대시보드에 단순 그래프만 존재, 인사이트 제공 없음
6. **보호자 소통 제한**: 알림톡은 일방향 발송만 가능, 양방향 소통 없음

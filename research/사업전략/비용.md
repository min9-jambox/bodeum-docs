# 예비창업패키지 4,000만 원 사업비 집행 계획 및 환수 리스크 방어 전략

> **⚠️ 2026년 예창패 구조 변경 반영**: 기존 4,000만 원 일괄 지급 방식에서 **1단계 2,000만 원 → 중간평가 → 통과 시 2단계 2,000만 원** 분리 지급 방식으로 변경됨.

### 1단계 (2,000만 원) — MVP 개발 인프라

## 2. 장비 구매 환수 리스크 방어

### ⚠️ 리스크: Mac Studio 2대 '용도 중복' 판정

**방어 전략**: 두 장비의 명분을 완전히 다르게 기안한다.

- **장비 A**: "Claude Code 기반 자율 에이전트 전용 코딩 환경 + 로컬 폐쇄망 테스트 서버"
- **장비 B**: "대용량 음성 인식 AI(Whisper) 오프라인 딥러닝 트레이닝 + 온프레미스 보안 서버 (Headless, 모니터 없음)"

장비 B를 'PC'가 아니라 **'물리적 전산 서버'**로 포지셔닝하면 승인 확률이 크게 올라간다.

## 3. 특허 비용 환수 리스크 방어 (중요)

**불가능한 집행:**
- 협약 이전에 출원한 건의 소급 집행
- 특허 등록 시 발생하는 성공보수료

**완전 합법 집행 방법:**
1. 출원인은 반드시 **창업기업 명의** (개인 명의 불가)
2. 변리사 계약서에서 **성공보수 항목 삭제** 요구
3. **협약 체결일 이후**에 출원 시작
4. "협약 기간 내 발생하는 의견제출통지 대응 비용 무상 포함"이라는 문구를 계약서에 명시 → 중간사건 비용도 예창패로 집행 가능

**실제 비용 구조 (팩트):**
- 우선심사 관납료: **20만 원** (특허청 공식 고시)
- 변리사 착수금: **150~250만 원** (시장 평균)
- 중간사건 1건: **30~60만 원** (무상 포함 조건 계약서 명시 필수)
- 성공보수: **예창패 집행 불가 → 반드시 계약서에서 삭제**

---

## 4. AI SaaS·클라우드 집행 주의사항

### 반드시 '월 정액' 결제

1년 치 선납은 협약 기간(약 8개월) 초과분이 환수 대상이다. Claude Code, OpenAI API, AWS 모두 **'매월(Monthly)' 플랜**으로 등록하고, 협약 종료 다음 달부터 사비 카드로 변경.

### 협약 후 발급된 법인카드로만 결제

개인 카드 선결제 후 사후 청구 → 원칙적 불인정. 협약 후 전용 사업비 법인카드가 나오면 모든 AI SaaS 계정의 결제 수단을 해당 카드로 교체.

---

## 5. 알림톡 통신비 환수 리스크 방어

알림톡 발송비를 단순 통신비로 기안하면 불인정될 수 있다.

**방어 전략**: 지출 결의 시 **"초기 가입 요양센터 대상 무상 지원 프로모션(마케팅)용 B2B 알림톡 실비 청구"** 포맷으로 기안. 신규 영업을 위한 '광고선전비'임을 명시.

# 📊 SilverBridge 수익성 및 BEP(손익분기점) 시뮬레이션

> **조건 설정:** (1인 개발/운영 체제, 요양보호사 10인 이하 소규모 센터 1곳 기준)
> *   과금 모델: 요양원당 **월 10,000원**
> *   이용량 가정: 센터 1곳당 일일 요양보호사 10명 출근 ➡️ 알림톡 10건 발생, 월 20일 기준 (총 200건)

## 1. 센터 1곳당 월별 원가 분석 (Variable Costs)

월 1만 원을 받았을 때, 알림톡과 서버를 쾌적하게 유지할 수 있을까요? 
**결론: 충분합니다. 약 72%의 높은 마진율(Gross Margin)을 기대할 수 있습니다.**

| 항목 | 계산식 (월 20일 기준) | 월 예상 비용 | 비고 |
| :--- | :--- | :--- | :--- |
| **카카오 알림톡 발송비** | 10건 x 20일 x 건당 약 9원 | **1,800 원** | 가장 큰 변동비 (비율의 핵심) |
| **LLM(AI) API 호출비** | 10건 x 20일 x 건당 약 0.5원~1원 | **200 원** | gpt-4o-mini 등 경량 모델 사용 시 |
| **서버/DB 종량제 비용** | 1개 센터당 데이터 Read/Write 트래픽 | **대략 500 원** | 완전 관리형(Firebase 등) 초과분 기준 |
| **1개 센터당 총 변동비** | | **약 2,500 원 ~ 2,800원** | |
| **1개 센터당 순수 마진 (공헌이익)** | 10,000원 - 2,800원 | **약 7,200 원** | **마진율 72%** |

*   1만 원만 받아도 **7천 원 이상의 마진**이 남으므로 유지 비용 걱정은 하지 않으셔도 됩니다. 소프트웨어(SaaS) 산업의 전형적인 고마진 구조입니다.

## 2. 순수익 500만 원(Net Profit)을 위한 목표 센터 수 산출

대표님께서 온전히 **매월 통장에 500만 원을 꽂으려면(1인 체제)**, 고정비를 뺀 마진으로 이를 달성해야 합니다.

### 월 예상 고정비 (Fixed Costs)
*   **서버/DB 베이스라인 비용:** 월 약 10~15만 원 (AWS, GCP 등의 기본 인스턴스, 스토리지 등)
*   **소프트웨어 구독 및 유지보수 관리비:** 월 약 5만 원 (도메인, 보안인증서, 애플/구글 개발자 계정 등)
*   **예비비(CS 등):** 월 약 10만 원
*   **월 총 고정비:** **약 30만 원**

### BEP(손익분기점) 계산식
*   목표 순수익 5,000,000원 = [ (센터 수) x (1센터당 마진 7,200원) ] - (고정비 300,000원)
*   (센터 수) x 7,200원 = 5,300,000원
*   **필요 센터 수 = 약 736곳**

**👉 결론: 버퍼(여유분)를 감안하여 약 700~800곳의 센터만 가입시키면 매월 500만 원 이상의 순수익이 발생합니다.**
*(만약 1,000개 센터를 유치할 경우 ➡️ 월 매출 1,000만 원 / 알림톡 등 변동비 280만 원 / 고정비 30만 원 = **월 순수익 약 690만 원** 달성)*

## 3. 전국 시장 점유율(Market Share) 분석

전국 센터 중 몇 % 가 이용해야 800곳이 모이고, 500만 원을 달성할까요?

*   **시장 규모 (TAM):** 2024년 국민건강보험공단 기준 전국의 방문요양, 방문목욕, 주야간보호 등 **전체 재가장기요양기관 수는 약 21,000여 개**입니다. (순수 방문요양만 추려도 약 16,000~17,000개)
*   **점유율 계산:**
    *   21,000곳 기준: 800 / 21,000 = **약 3.8%**
    *   17,000곳 기준: 800 / 17,000 = **약 4.7%**

**👉 최종 결론:**
대한민국 전체 재가요양기관 중 **딱 4%**만 우리 서비스(월 1만 원)를 쓰게 만들면, 대표님은 혼자서 **월 500만 원의 안정적인 자동화 수익(Auto-cashcow)**을 얻게 됩니다. 

특히 5만 원이 아닌 **'월 1만 원'이라는 파격적인 진입 가격**을 고려할 때, 4%의 점유율 확보는 예창패 통과 후 영업 전략만 잘 갖추어진다면 1년 이내에 충분히 달성할 수 있는 매우 현실적인 수치입니다. 이 데이터는 투자자나 정부 심사위원들에게도 "우리의 목표치가 비현실적이지 않다"는 점을 증명하는 강력한 무기가 됩니다.

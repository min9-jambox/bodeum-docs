## 2. 컬러 시스템

### 2.1 Primary Colors

| 이름 | HEX | RGB | HSL | 용도 |
|------|:---:|:---:|:---:|------|
| **Bodeum Blue** | `#4A7FD4` | 74, 127, 212 | 217°, 62%, 56% | 주 브랜드 컬러. 앱 바, 주요 CTA 버튼, 로고 |
| **Bodeum Deep** | `#2B5EA6` | 43, 94, 166 | 215°, 59%, 41% | 호버/프레스 상태, 헤더, 강조 텍스트 |
| **Bodeum Light** | `#E8F0FE` | 232, 240, 254 | 218°, 85%, 95% | 배경 하이라이트, 선택 상태, 뱃지 배경 |

> **왜 Blue인가?** — 신뢰(Trust)와 전문성(Expertise)을 전달하는 색상. 의료/헬스케어 B2B SaaS에서 가장 보편적이며, 고령 사용자의 시인성(가독성)이 높다.

### 2.2 Secondary Colors — Warm Accent

| 이름 | HEX | RGB | 용도 |
|------|:---:|:---:|------|
| **Warm Coral** | `#F28B6E` | 242, 139, 110 | 따뜻함 강조. 온보딩, 성공 상태, 포인트 아이콘 |
| **Warm Light** | `#FFF0EB` | 255, 240, 235 | Coral 배경, 알림 뱃지 배경 |

> **왜 Coral인가?** — "보듬다"의 따뜻함을 시각화. 차가운 Blue만으로는 돌봄 서비스의 감성을 전달하기 어렵다. 주의를 끌어야 하는 핵심 액션(녹음 시작, 알림 등)에 사용.

## 3. 타이포그래피

### 3.1 서체

| 용도 | 서체 | 비고 |
|------|------|------|
| **한글 본문** | Pretendard | 가변 폰트. 웹+앱 통일. 무료 오픈소스 |
| **영문/숫자** | Pretendard | 한/영 혼용 시 통일된 비율 유지 |
| **코드/기술 문서** | JetBrains Mono | 개발 문서, 코드 블록 |

> **왜 Pretendard인가?** — 한국어 가독성 최적화. Apple SD Gothic Neo, Noto Sans KR 대비 시인성이 뛰어나며, 가변 폰트로 다양한 weight를 로딩 부담 없이 사용 가능. 2024~2026년 한국 웹/앱 서비스에서 가장 많이 사용되는 한글 서체.

### 3.2 타입 스케일

| 이름 | 크기 (rem/px) | Weight | 용도 |
|------|:---:|:---:|------|
| **Heading 1** | 1.875 / 30px | Bold (700) | 페이지 타이틀 |
| **Heading 2** | 1.5 / 24px | SemiBold (600) | 섹션 타이틀 |
| **Heading 3** | 1.25 / 20px | SemiBold (600) | 카드 타이틀 |
| **Body Large** | 1.125 / 18px | Regular (400) | 본문 강조 |
| **Body** | 1 / 16px | Regular (400) | 기본 본문 |
| **Body Small** | 0.875 / 14px | Regular (400) | 보조 텍스트, 캡션 |
| **Caption** | 0.75 / 12px | Medium (500) | 레이블, 뱃지, 타임스탬프 |

#### 고령 사용자 접근성 가이드

| 항목 | 규칙 |
|------|------|
| **최소 본문 크기** | 앱: 16px 이상 (모바일), 웹: 14px 이상 |
| **최소 터치 타겟** | 앱: 48×48dp (구글 권장), 주요 버튼: 56×56dp |
| **줄 간격** | 본문 1.6 이상 (기본 1.65) |
| **글자 간격** | -0.01em (Pretendard 기본) |
| **대비 비율** | 본문: 4.5:1 이상, 대형 텍스트: 3:1 이상 (WCAG AA) |

## 5. 로고 가이드

### 5.1 로고 구성

```
┌────────────────────────────────────────┐
│                                         │
│   🤲  보듬        ← 워드마크 (한글)      │
│       Bodeum      ← 서브 워드마크 (영문)  │
│                                         │
│   심볼: 두 손이 감싸는 형상 (하트+손)     │
│   컬러: Bodeum Blue (#4A7FD4)           │
│                                         │
└────────────────────────────────────────┘
```

### 5.2 로고 형태

| 형태 | 사용 환경 | 최소 크기 |
|------|----------|----------|
| **심볼 + 한글 워드마크** (가로) | 앱 스플래시, 웹 헤더, 명함 | 높이 32px |
| **심볼 + 영문 워드마크** (가로) | 영문 마케팅, 해외 문서 | 높이 32px |
| **심볼만** | 앱 아이콘, 파비콘, 작은 공간 | 24×24px |
| **워드마크만** (한글) | 텍스트 환경, 서명 | — |

### 5.3 로고 사용 규칙

| 규칙 | 설명 |
|------|------|
| **여백** | 심볼 높이의 50% 이상 사방 여백 확보 |
| **배경** | 흰색/밝은 배경: 기본 컬러 사용. 어두운 배경: 흰색 버전 사용 |
| **금지** | 비율 변경, 그림자 추가, 컬러 변경, 배경 대비 낮은 곳 사용 금지 |

## 1. WBS 코드 체계

```
[Phase].[영역].[순번]

Phase 0  = 0     (프로젝트 초기 설정)
Phase 1  = 1     (MVP — 1단계 P0)
Phase 1B = 1B    (1단계 P1)
Phase 2  = 2     (성장)
Phase 3  = 3     (확장)

영역 코드:
  INF = 인프라·DevOps        COM = 공통 모듈
  AUTH = 인증 서비스          CARE = 돌봄 서비스
  AI  = AI 파이프라인         NOTI = 알림 서비스
  E2E = 통합 테스트           APP = 모바일 앱
  WEB = 웹 백오피스           ADMIN = 어드민
  OPS = 시범 운영             CJ = 상담일지
  PAY = 결제                  GEO = 지오펜싱
  DASH = 대시보드             SCH = 일정
  EVAL = 기초평가             ATT = 근무일지
  RPT = 리포트                MEET = 회의록
  ERP = 공단 연동             BILL = 본인부담금
  PAYROLL = 직원 급여         GUIDE = AI 가이드
  TREND = 건강 트렌드         MULTI = 멀티 기관
  BIZ = 사업(특허 등)
```
